# 🤖 AI-Based Hiring Prediction System

An end-to-end Machine Learning project that predicts whether a candidate is likely to be **Hired** or **Rejected** based on resume information such as skills, work experience, education, certifications, projects, and salary expectations.

> 🚧 **Project Status:** In Progress

---

## 📌 Project Objective

The goal of this project is to simulate an AI-powered resume screening system used in modern recruitment processes. The model analyzes candidate information and predicts hiring outcomes to assist recruiters in making data-driven decisions.

---

## 📂 Dataset Features

The dataset contains synthetic resume data with the following attributes:

- Resume ID
- Candidate Name
- Skills
- Experience (Years)
- Education
- Certifications
- Job Role
- Salary Expectation
- Projects Count
- Recruiter Decision (Target Variable)
